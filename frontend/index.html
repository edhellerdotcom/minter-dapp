Welcome to the ChicagoPizzaRDD NFT Bakery

The ChicagoPizzaRDD NFT Project is a project that aims to add more Chicago Pizza awareness to the NFT Collecting Community. Check RealDeepDish.com/nft for info on current and upcoming tokens being baked in our blockchain ovens.

To get started:


DESKTOP/LAPTOP USERS: Press the "Install MetaMask!" button or go to
metamask.io/download
to install the extension in your browser.


Then press the Connect button to connect your MetaMask wallet.



MOBILE USERS:

If you are on a mobile device, you will first need to install the
MetaMask mobile wallet app.


Set up your wallet (In the US, you may need to go through some extra steps),
add (or switch to) the Polygon Mainnet network,
and load up some Polygon/MATIC tokens in there.
Then, open this webpage, which is realdeepdish.netlify.app INSIDE the MetaMask browser.

HOW DO I OPEN A WEB BROWSER INSIDE METAMASK APP?
On the top left of the MetaMask screen, you will see 3 horizontal lines (aka the hamburger). Press that, then select BROWSER.
Then either type in realdeepdish.netlify.app in the homepage search bar. If you see "no open tabs" then press the + (plus) to open one first.

OKAY, I GOT THIS PAGE LOADED; NOW WHAT?
Press the "Connect" button, then go to the 3 dots (on the bottom right) and RELOAD the page to get the minting interface up. Then you should be able to try minting.

For the first token series, you'll need at least 15 MATIC per token, not including 'gas fees', which you're going to see a lot when making transactions on blockchain networks.


WHERE DO I BUY Polygon MATIC?
MetaMask has a way to purchase MATIC tokens in the app. You might find tokens with less fees at places like MoonPay.com
